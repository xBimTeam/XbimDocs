IfcTimeSeriesSchedule
Definition from IAI: The IfcTimeSeriesSchedule defines a time-series that is applicable to to one or more calendar dates. It typically contains a periodically repetitive time series used to define the schedule, facilitating the capture of hours of operation, occupancy loads, etc. HISTORY: New entity in Release IFC2x Edition 2.
