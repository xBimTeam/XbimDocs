Version 1.0.0.0

Version 3.0 was released on 5th January 2015.

xBIM Version 3

  • This contains a major upgrade to the geometry engine, allowing authoring as well as presentation of IFC geometries. All boundary representation solid, shell, face, edge and vertex behaviours are implemented

    All entities in Ifc 2x3 are now supported

    The web viewing components have been written and we now provide a high performance standalone javascript client that is tuned for WebGL

    We have moved from the Codeplex platform to GitHub by popular demand of the developers

    All packages are now distributed by GitHub
